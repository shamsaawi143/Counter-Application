
Counter Application

A simple and interactive Counter Application built using HTML, CSS, and JavaScript.
This project demonstrates basic DOM manipulation, event handling, and dynamic UI updates in a clean and modern interface.

📌 Overview

The Counter Application allows users to:

Increase the counter value
Decrease the counter value
Reset the counter to zero

It is a beginner-friendly project designed to strengthen JavaScript fundamentals and improve understanding of user interactions in web development.

✨ Features
✅ Increment (+) button
✅ Decrement (−) button
✅ Reset button
✅ Real-time value updates
✅ Clean and responsive design
✅ Beginner-friendly code structure
🛠️ Technologies Used
HTML5
CSS3
JavaScript (Vanilla JS)

🎯 How It Works
JavaScript selects the counter display element.
Event listeners are attached to buttons.
Clicking a button updates the counter value dynamically.
The UI reflects changes instantly without refreshing the page.
